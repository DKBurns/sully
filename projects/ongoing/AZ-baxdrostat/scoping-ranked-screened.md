## Selection Notes

- **Known sources already cover** MI/stroke/HF/CVD death/QALYs/utilities. 5 new papers should prioritize **gap endpoints**: eGFR trajectories, uACR, CKD progression, MCI/dementia, DR.
- **Ranks 1 + 3** (CRIC 33161774, CRIC 26711738) = same cohort, complementary analyses but overlapping data.
- **Rank 2** (UKPDS 38) uniquely covers DR + macrovascular — T2D-only but model handles diabetic at baseline.
- **Rank 5** (Danish uACR 40398961) = largest N, only strong uACR-stratified outcome source.
- **Rank 8** (SPRINT-MIND 32591439) = only strong source for adjudicated MCI/dementia in HTN.
- Balanced 5-paper set covering max endpoints: e.g. {1, 2, 4 or 5, 8, 9 or 11}.
